Greenly - Plataforma para conectar ONGs con Donantes (7mo ciclo)

Greenly es una aplicación web desarrollada durante el séptimo ciclo de mi carrera, cuyo propósito es conectar organizaciones no gubernamentales que apoyan a comunidades vulnerables con personas dispuestas a donar. La plataforma permite crear publicaciones, interactuar mediante comentarios y realizar donaciones con seguimiento en tiempo real. Para su desarrollo se utilizó Spring Boot en el backend y Angular en el frontend, logrando una experiencia intuitiva y escalable.

    Características principales

  • Interfaz intuitiva: Los usuarios pueden visualizar campañas, realizar donaciones y dejar comentarios directamente desde la plataforma.
  • Seguimiento de donaciones: Cada usuario tiene acceso al historial y estado actual de sus donaciones.
  • Desarrollo Full Stack: Implementación del backend con Spring Boot y el frontend con Angular, siguiendo principios de arquitectura escalable y modular.
