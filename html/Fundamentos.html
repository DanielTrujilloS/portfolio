Sistema de Recopilación y Análisis de Datos para la empresa Laive(5to ciclo)

Este proyecto fue desarrollado durante el quinto ciclo de mi carrera, utilizando C# como lenguaje de programación y MySQL para la conexión con la base de datos. El objetivo principal era solucionar un problema empresarial en una compañía líder del mercado peruano, para lo cual elegimos Laive. Nuestra solución permitía el ingreso y gestión eficiente de datos, asi como la ejecución de reportes específicos para optimizar los procesos internos de la empresa.

    Caracteristicas principales

  • Prototipado:Se desarrolló los mockups en Adobe XD, para una mejor implementación al momento de programar.
  • Efectividad : Logramos ejecutar el codigo, registrar los datos de los clientes, y generar reportes de forma eficiente y correcta.
  • Interactividad:Uso de C# para optimizar la integración con SQL Server, permitiendo una gestión eficiente y estructurada del registro de datos, garantizando precisión, seguridad y alto rendimiento en las operaciones.
